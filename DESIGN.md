# DESIGN.md

## 气质与意象
黑暗机械深渊中闪烁的黄色警示灯——锈蚀金属面板、全息投影数据流、布线走廊中透出的幽绿光芒。一切精密而冰冷，唯有高亮数据打破沉寂。

## 配色方案
| Token | 色值 | 意象来源 |
|-------|------|----------|
| 主背景 | `#0F2620` | 未见天日的 Mechanicus 舰船内部——暗绿金属壁板 |
| 辅背景 | `#1A1A1A` | 纯黑暗室——数据终端熄屏后的虚空 |
| 高亮色 | `#E6C200` | Omnissiah 圣光——警告灯/关键数据/按钮边框 |
| 主文本 | `#F0F0F0` | 全息投影文字 |
| 次文本 | `#B0B0B0` | 终端灰度信息 |
| 卡片背景 | `#162B24` | 舱壁面板——比主背景略亮，可区分层次 |
| 边框 | `#E6C200/20%` | 黄色微光描边 |

## 字体排版
- 字体族：Inter（Google Fonts .cn 域名加载），sans-serif 回退
- 标题：粗体 700，大写字母间距 `tracking-wide`
- 正文：常规 400，行高 1.75
- 数据/代码：Roboto Mono

## 动效与交互
- 悬停：黄色边框/背景亮度提升，`transition 200ms ease`
- 卡片：hover 时 `translateY(-2px)` + shadow 加深
- 按钮：hover 背景变亮 10%，active 按下缩放 0.98
- 页面切换：无过度动画，保持信息密度优先

## 页面结构
- 顶栏：固定定位，半透明深色背景 + `backdrop-blur`
- 内容区：max-width 1200px 居中
- 卡片：暗背景 + 黄色细边框 + 微阴影
- 表格：简洁风格，稀有材料列用黄色高亮

## 设计禁忌
- 禁止使用紫色/蓝色系——这不是 Sci-Fi 而是 Grimdark
- 禁止圆角过大——最大 `rounded-lg`，机械风格偏好直角或小圆角
- 禁止渐变背景——纯色 + 边框层次
- 禁止使用任何游戏官方 LOGO / 原画 / UI 截图
- 禁止使用 "Official" "Authorized" "Partnered" 等词汇
